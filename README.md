# 💻 Projeto Trimestral de Banco de Dados

Projeto desenvolvido individualmente para a disciplina de Banco de Dados, utilizando o **MySQL Workbench**.

O foco do trabalho foi criar um modelo lógico e físico para representar as tabelas e os relacionamentos de um sistema fictício, além de gerar o código SQL correspondente.

---

## 🧠 Etapas do Projeto

- Criação do modelo lógico com as entidades e atributos principais  
- Definição dos relacionamentos entre as tabelas  
- Modelagem física no MySQL Workbench  
- Geração do código SQL  
- Execução no banco de dados com inserção de dados e consultas de teste
